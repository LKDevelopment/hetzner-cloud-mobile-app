Rescue Modus Einstellungen RESCUE

Wenn Sie das Rescue-System aktivieren, wird Ihr Server beim nächsten Neustart automatisch über das Netzwerk in das Rescue-System booten. Sie können das Rescue-System anschließend solange verwenden wie Sie es benötigen.

Der Auftrag für das Rescue-System bleibt 60 Minuten aktiviert. Wenn Sie danach Ihren Server neu starten, wird wieder Ihr gewohntes System von der Festplatte gestartet.

 ROOT-PASSWORD

Es wird ein neues Root-Passwort auf Ihrem Server gesetzt. Falls Sie den QEMU Guest Agent vom Server entfernt haben, wird diese Aktion fehlschlagen.
