Power Einstellungen POWER

„Herunterfahren“ schickt ein ACPI signal an Ihren Server. In unserer Standardkonfiguration fährt Ihr Server geordnet herunter.

„Power off“ schaltet Ihren Server hart aus. Diese Aktion kann zu Datenverlust führen.

Bitte beachten Sie, dass ausgeschaltete Server weiterhin berechnet werden. In unserer FAQ finden Sie weitere Details hierzu.

 POWER RESET

„Power Reset“ löst einen harten Reset Ihres Servers aus. Diese Aktion kann zu Datenverlust führen.
